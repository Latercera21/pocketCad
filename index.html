PocketCad

Moldería digital para la industria textil

PocketCad es una herramienta para crear, editar y preparar moldes digitales.
Permite trabajar con piezas, curvas, cortes, simetrías, dirección de hilo y exportación de proyectos.

Guía rápida de herramientas

◻️ Crear polígono

Crea una nueva pieza base.

🖊️ Crear línea

Dibuja líneas de construcción o corte.

📌 Editar vértice

Permite mover vértices manualmente. Incluye un candado para fijar el vértice seleccionado y evitar movimientos libres. También permite aplicar desplazamientos horizontales y verticales mediante valores numéricos.

🧲 Snap

Activa el ajuste automático entre puntos y elementos cercanos para facilitar alineaciones precisas.

⚫ Agregar vértice

Inserta nuevos puntos en una arista.

➖ Eliminar vértice

Elimina puntos seleccionados de una figura manteniendo la continuidad de la geometría.

📏 Cambiar longitud

Modifica la longitud exacta de una arista seleccionada mediante una medida introducida en centímetros.

〰️ Curvar arista

Convierte una línea recta en curva.

📐 Curva a recta

Convierte una curva nuevamente en línea recta.

📖 Desdoblar

Crea automáticamente la parte opuesta de una pieza simétrica.

✋ Mover figura

Desplaza una pieza completa.

🔒 Cerrar figura

Marca una pieza como cerrada para completar su contorno. Algunas operaciones requieren una figura cerrada para funcionar correctamente.

⛓ Unir curvas

Combina curvas consecutivas para formar un trazado continuo más limpio y uniforme.

🔄 Rotar figura

Gira una figura según el ángulo indicado.

⧉ Duplicar figura

Crea una copia exacta.

🪞 Reflejar figura

Genera una versión simétrica.

⇄ Costura / Desfase

Genera una línea paralela a una arista seleccionada. Se utiliza para crear márgenes de costura, dobladillos y separaciones controladas. El valor se introduce en centímetros.

✂️ Cortar figura

Divide una pieza mediante líneas de corte.

🧵 Dirección de hilo

Define la orientación del tejido para corte y confección.

🗑️ Eliminar figura

Elimina completamente una pieza seleccionada del proyecto.

🔐 Bloquear figura

Impide modificaciones accidentales sobre una pieza. Una figura bloqueada no puede moverse ni editarse hasta ser desbloqueada nuevamente.

💾 Exportar JSON

Guarda el proyecto actual.

🖼️ Exportar SVG

Exporta las piezas en formato SVG para utilizarlas en otros programas de diseño, corte o anidado.

📂 Importar JSON

Carga un proyecto guardado.

↩️ Deshacer

Revierte la última acción realizada.
